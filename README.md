# Javascript 멘토링 소스코드 허브

- 이 리포지토리는 멘토링 세션에서 사용되는 실습 자료와 예제 코드를 공유하는 공간입니다.
- 멘토와 멘티 모두가 쉽게 접근하고 활용할 수 있도록 구성되어 있습니다.

## 목적

- 멘토링 세션에서 다루는 주요 개념의 실제 구현 예시 제공
- 멘티들의 자기주도 학습을 위한 추가 자료 제공
- 관심 있는 주제의 폴더를 탐색해보세요.
